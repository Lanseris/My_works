Пока совсем простая и не до конца работающая
Написана для демонстрации работы с WPF, SQL, и попытки использования веб-сервиса (ASP.NET).

Ранее в коде этой же программы была реализована работа напрямую с SQL базой данных, после чего,
 проект был разделён на две части: клиентскую часть и API-сервис.

На данный момент не работает полностью, да и проверить то, что работает, обычному пользователю будет проблематично, 
т. к. для проверки работы того что работает у меня, необходимо будет запустить у себя SQL сервер,
 запустить на нём код создания самой базы данных(который находится в папке DataBase), заполнить его хоть чуть чуть,
прописать в трёх классах (Скрин №1)API сервиса путь к базе данных вручную вот сюда (Скрин_№2),
 после чего как минимум API сервис должен будет работать и
 можно будет проверить его работу при помощи  удобного расширения для браузера "Restlet Client" 
путём написания запросов в ручную (Скрин №3)
запросы :
 - api/departments - список всех отделов с их сотрудниками
 - api/employees - список всех сотрудников
 - api/DepVsEmplLink/(ID любого отдела) - список ID сотрудников пренадлежащих отделу, ID которого был указан в запросе

Если у вас получилось, и вы не удвлетворены, можете попробовать запустить клиентское приложение, но т. к. приложение
находится в тестовом режиме ничего кроме пустой консоли вы не увидите, НО, если увидите и не вылезет ни одной ошибки ->
программа успешно обратилась к API сервису и считала все необходимые данные, 
поверить это можно путём постановки "точки остановы" здесь (Скрин №4).

А WPF-ом можно полюбоваться пока в самой Visual Studio там остался код, связанный с рабочей версией программы,
 контактирующей напрямую с БД.
