1. Проверка ввода логина удвлетворяющего условию с использованием регулярных выражений
  строка от 2-х до 10-ти символов, содержащая только буквы или цифры, и при этом цифра не может быть первой.

2. Решение различных задач при помощи регулярных выражений
а) Вывести только те слова сообщения, которые содержат не более чем n букв;
б) Удалить из сообщения все слова, которые заканчиваются на заданный символ;
в) Найти самое длинное слово сообщения;
г) Найти все самые длинные слова сообщения.

3. Метод для двух строк, определяющий, является ли одна строка перестановкой другой без учёта регистра.
 Реализовано двумя способами: методами С#, LINQ

4. Есть список учеников из записей такого вида:
Иванов Петр 4 5 3
Программа выводит информацию об учениках, имеющих худшие показатели. Если среди остальных есть ученики,
набравшие тот же средний балл, что и один из трех худших, то информация о нём так же выводится.
